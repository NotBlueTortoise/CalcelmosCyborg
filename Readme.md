# Calcelmo's Cyborg
An NSFW mod for Skyrim.

Centered around the transformation into a cyborg to an extent controllable by the player.
The story will, as the title suggests, be centered around Calcelmo in Markarth.

## Requirements
### Hard Dependencies
 - [Devious Devices Integration 4.0](https://www.loverslab.com/files/file/371-devious-devices-integration/)
### Soft Dependencies
 - SKSE + SkyUI - If you want access to the MCM.
### For Developers
 - SKSE Sources
 - [SkyUI SDK](https://github.com/schlangster/skyui/wiki#skyui-sdk)
 - Devious Devices Integration Sources and all sources it requires

## Building and Installing
### Building the project
 - Clone this project using Git.
 - Install the regular requirements as well as the ones 'For Developers' (see above).
 - Set up a way to compile the scripts (either a batch file or an IDE like Atom or Sublime Text with the relevant plugin).
 - Compile the scripts.
### Installing the result
 - Pack the *scripts*, *meshes* and *textures* folders into a BSA.
 - Place the ESP and BSA files and the CalienteTools folder into an archive file (.zip / .7z) and install with the mod manager of your choice.
